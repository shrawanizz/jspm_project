LRU Cache
Design a data structure for a Least Recently Used (LRU) cache. Implement LRUCache with get and put in O(1).
Example
Input: ["LRUCache","put","get"]
    [[2],[1,1],[1]]
Output: [null,null,1]
Hints
Think HashMap + Doubly Linked List. HashMap gives O(1) lookup. DLL maintains recency order with O(1) insert/remove.
Use sentinel head/tail nodes. On access, move node to front. Evict from tail (LRU). Avoids null checks.
Constraints
1 ≤ capacity ≤ 3000
0 ≤ key ≤ 10⁴
≤ 2 × 10⁵ calls
